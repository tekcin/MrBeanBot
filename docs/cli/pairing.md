# `MrBeanBot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
MrBeanBot pairing list whatsapp
MrBeanBot pairing approve whatsapp <code> --notify
```
